enum LED_Command

Enumeration representing LED operation types for queued commands. 

      This enum defines the possible actions that can be queued
      to control an LED (turn it on, off, or toggle its state).
Enumerator

◆ LED_RTOS_GetState()

bool LED_RTOS_GetState ( LED_ColorValueType color,
bool * state )

Gets the current state of a specific LED. 

      This function retrieves the state of the LED corresponding to the given color.
      It uses a mutex to ensure thread-safe access. If the mutex cannot be obtained,
      the function returns false and the state is not updated.
Parameters
colorThe color of the LED to query.
statePointer to a boolean variable to store the LED state. Will be set to true if the LED is ON, or false if OFF.
Return values
trueIf the state was successfully retrieved.
falseIf the mutex could not be acquired or state retrieval failed.
