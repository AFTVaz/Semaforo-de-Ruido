◆ WIFI_Init()

void WIFI_Init ( WIFI_Config * config)

Initializes the Wi-Fi module and optionally related services. 

   This function stores the SSID and password used for connection,
   creates synchronization primitives, and starts the internal Wi-Fi task.
   It can also initialize NTP and/or MQTT services depending on the configuration.
Parameters
configPointer to a WIFI_ConfigType structure containing the SSID, password, and service initialization options.
Note
The config pointer and its fields must not be NULL. SSID and password strings must remain valid during initialization.

Parameters

   performed without interleaving.

Return values
trueIf the transaction request was queued successfully.
falseIf the MQTT is currently busy or the queue is unavailable.
