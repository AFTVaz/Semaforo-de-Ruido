◆ PINCFG_FunctionEnum

enum PINCFG_FunctionEnum

Enumeration of possible functions for a pin.

Each pin can be configured for a specific function such as GPIO, UART, SPI, I2C, PWM, ADC, etc. The numeric value corresponds to the PINSEL register configuration.
