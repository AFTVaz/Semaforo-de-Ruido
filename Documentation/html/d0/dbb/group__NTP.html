NTP_StateValueType NTP_GetState ( void )

Retrieves the current state of the NTP client. 

   Returns the internally tracked NTP client state, ensuring thread safety
   by using a FreeRTOS mutex.
Returns
The current state as a value of NTP_StateValueType.
Return values
NTP_IDLEIf the client is idle or ready for a new request.
NTP_PROCESSINGIf the client is processing a time request.

◆ NTP_GetTimestamp()

bool NTP_GetTimestamp ( time_t * timestamp)

Retrieves the latest timestamp received from the NTP server. 

   This function returns the converted "struct tm" (UTC/GMT) if a valid
   timestamp is available. It resets the internal state to "NTP_IDLE"
   after a successful retrieval.
Parameters
timestampPointer to a "time_t" where the result will be stored.
Return values
trueIf a valid timestamp was available and "timestamp" was populated.
falseIf the timestamp is not available or "timestamp" is NULL.
Note
This function is non-blocking and should be used after calling NTP_RequestTime. Use NTP_RequestAndWait for a blocking alternative.

◆ NTP_Init()

void NTP_Init ( void )

Initializes the NTP module and creates the background task. 

   This function sets up the mutex used to protect access to shared NTP state
   and creates the FreeRTOS task responsible for handling NTP time synchronisation.
Note
Must be called once before any other NTP-related function.

◆ NTP_RequestAndWait()

bool NTP_RequestAndWait ( time_t * timestamp)

Sends an NTP request and blocks until a response is received. 

   This function combines the request and polling logic to retrieve
   the timestamp in a blocking manner, useful in cases where the application
   flow depends on synchronized time.
Parameters
timestampPointer to a "time_t" where the result will be stored.
Return values
trueIf the timestamp was received and "timestamp" was successfully populated.
falseIf an error or if "timestamp" is NULL.

◆ NTP_RequestTime()

void NTP_RequestTime ( void )

Sends a time request to the configured NTP server. 

   This function notifies the NTP task to start the process of connecting,
   sending the NTP packet, and waiting for a valid response.
Note
This function is non-blocking needs a follow up call of NTP_GetTime. Use NTP_RequestAndWait to block until time is available.
