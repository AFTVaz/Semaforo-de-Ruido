◆ BUZZER_Command

enum BUZZER_Command

Enumeration representing buzzer command types. 

   This enum defines the possible actions that can be sent
   to the buzzer task, such as turning it ON, OFF or triggering an alarm.
Enumerator
BUZZER_CMD_ON 

Turn the buzzer ON.

BUZZER_CMD_OFF 

Turn the buzzer OFF.

BUZZER_CMD_ALARM 

Trigger an alarm for a specific duration.
