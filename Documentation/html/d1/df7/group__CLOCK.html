◆ CLOCK_GetSeconds()

time_t CLOCK_GetSeconds ( void )

Gets the current RTC time in seconds since 1970 (Epoch). 

   Returns the current time value in seconds since the Unix Epoch (1970-01-01).
   Read access is protected by a mutex to prevent interference from other tasks.
Returns
Current time in seconds since 1970-01-01.
Return values
0If the RTC is at the initial date or not initialized.
>0Valid time value.
Note
The function blocks execution until the mutex is available for safe reading.

◆ CLOCK_Init()

void CLOCK_Init ( void )

Initializes the RTC module and creates the control mutex. 

   This function checks if the RTC mutex has been properly created, and if not,
   it creates it before initializing the RTC. The mutex is used to protect
   concurrent access to the RTC.
Note
This function must be called before accessing any RTC-related function.
